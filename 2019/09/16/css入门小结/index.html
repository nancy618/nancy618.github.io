css入门小结

初识css相信很多小伙伴和我一样，只了解它的一些基础的东西，然鹅很多一知半解。但随着学习日程的更近，对它了解的加深。css通俗的来讲就是给代码化妆，你喜欢的样子它都有。废话不多说，总结了一些近日学习的笔记，分享给小伙伴~

首先最重要的一点是css的书写规范，任何语言它强调的是代码规范，记得上第一节课时老师对我们提的要求就是编码的规范，给我们讲了一个事例，阿里巴巴的员工进去干活之前首先就是写注释。假如写的乱七八糟，后期维护起来也很费劲。所以从一开始养成良好习惯，记得注释和规范,代码格式同样属于规范的一部分，要想作为一名优秀的前端程序员，就得好好学习如何提高代码的可读性和可维护性~

CSS文本样式：font-size:字号大小、font-family:字体、font-weight:字体粗细、font-variant:变体、font-style:字体风格…..等等，用过了就知道啦。

注释：/* 这是注释 / **CSS里面用到的注释只能用 /\ */,要不然会影响到下面代码的执行，CSS正确注释的快捷键为shift+ctrl+/。**

一、css的选择器，它一共分七种（css的应用范围的规则）包括：

标签/类型选择器（元素选择器“a”）

类选择器 “.” （.翻译过去的意思就是class类的意思）

id选择器（#）PS：id选择器与类选择器的区别–类选择器是可以重复多次使用，类似于人名可以n+,而id是对应的。

伪类选择器、伪元素选择器、属性选择器、

组合选择器（如果多个元素有相同的样式，将选择器写到一起并用逗号隔开）。

二、transition过渡（transition:过渡的属性、花费时间、运动曲线-默认是ease）

eg:transition:width 0.6s ease-in 0s; (长宽都能设定）

还有好玩的就是可以控制速度达到渐变的一种效果

eg:transition：background-color 2s; （时间过渡引起渐变效果）

PS：transition是写在div里不是hover里面。

三、transform 变换（动画）

学习了近一周的CSS，看完基础课程后，动手敲了一个小黄人的案例，发现了不少有趣的东西，它的布局和动画效果。经验还是得来自平时学习积累和实践，对于我这种捡了芝麻丢了西瓜的学习方式，自己值得深刻去反思。
