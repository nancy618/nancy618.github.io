JavaScript高级（学习笔记）

首先弄清楚目标：理解面向对象开发思想、掌握JavaScript面向对象开发相关模式、掌握JavaScript中使用正则表达式。

特点：1、解释执行、2、灵活 3、头等函数 4、执行环境（宿主环境）

组成：ECMAScript - 语法规范

​ 变量、数据类型、类型转换、操作符

​ 流程控制语句：判断、循环语句

​ 数组、函数、作用域、预解析

​ 对象、属性、方法、简单类型和复杂类型的区别

内置对象：Math、Date、Array，基本包装类型String、Number、Boolean

来简单概括一下它们用途

来介绍一下对象，然鹅~这里说的不是大猪蹄子，也不是象棋中的对象。 // 当然啦有对象还是超级幸福的一件事 ~ 手动给木木宝贝笔芯 (๑′ᴗ‵๑)❤

咳咳~说正事，是面向对象中的 “对象”。

辣么啥是面向对象呢？

简单地说是过程式代码的一种高度封装，目的在于提高代码的开发效率和可维护性。而我们所学的JavaScript 就是面向对象编程，简称OOP （是一种编程开发思想）。

它的特征：封装性、继承性、（多态性）抽象

在学习的过程中跟着老师做了一个贪吃蛇小游戏，还是非常有意思的，除了敲代码敲到手软。

在开发过程中用到了很多方法，包括继承和对象拷贝、闭包、自调用函数的参数等等，先说一下调用。

call （）改变函数的中的this，并直接调用，apply的应用，bind（）不会调用方法，只会返回一个新的函数和复制。

闭包：在一个作用域中可以访问另一个作用域的变量。

递归：函数自己调用自己。

对象的拷贝：又分为深拷贝和浅拷贝（顾名思义浅或深层复制）。

正则表达式（独立存在与JavaScript没有半毛钱关系）它的限定符好多鸭，记不住还是得看文档：https://blog.csdn.net/qq_40195958/article/details/78962334
