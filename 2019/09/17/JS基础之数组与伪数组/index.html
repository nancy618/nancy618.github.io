JS基础之数组与伪数组

数组的本质

本质上，数组属于一种特殊的对象。typeof运算符会返回数组的类型是object。

而js中数组的本质：

人类理解：数组就是数据的有序集合；

js理解：数据就是原型链中有 Array、prototype的对象；

目前知道的伪数组有：

Arguments对象、document.qutryselectAll( ‘div’)

伪数组和数组的区别：

真数组的长度是可变的，是内置的属性，伪数组的长度不可变，是自定义的一个属性 。真数组可以使用数组中的方法 ，伪数组不可以实用数组中的方法 。真数组可以访问数组的方法 伪数组不能直接访问数组的方法。

数组的特殊性体现在，它的键名是按次序排列的一组整数（0，1，2…）。

五个false值：0、NaN、字符串空值 ‘ ‘ 、undefined、null

function 和 Function 区别

Function是一个构造函数，用于动态构建一个函数。构造名称必须和类名保持一致

function是关键字，通过function生成的函数实际上是一个Function对象实例。

PS：而所有的function 是继承了Function的属性。

数组的API：

Array.prototype.concat

Array.prototype.sort

Array.prototype.toString

Array.prototype.join

Array.prototype.map

Array.prototype.filter

Array.prototype.reduce
